# 🌾 AgroPulse - Crop Price Prediction System

AgroPulse is a machine learning-based web application designed to forecast agricultural commodity prices using real-time data from Indian government APIs. The platform helps farmers, traders, and stakeholders make data-driven decisions.

## 🚀 Features
- 📊 Real-time crop price visualization
- 📈 Price trend analysis
- 🤖 Machine learning-based forecasting
- 🌐 Interactive web interface using Streamlit
- 📍 Focus on Maharashtra agricultural markets

## 🛠️ Tech Stack
- Python
- Streamlit
- Machine Learning (Regression Models)
- Pandas, NumPy, Matplotlib
- Government Agri APIs

## 🎯 Objective
To provide an intelligent system that predicts future crop prices and reduces uncertainty in agricultural planning.

## 📌 Future Enhancements
- Integration of deep learning models (LSTM)
- Weather data analysis
